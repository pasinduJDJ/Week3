Django Blog Web CRUD App

Features:
- View all posts
- View single post
- Add / Edit / Delete post
- Success messages using Bootstrap alerts

Setup:
- git clone https://github.com/pasinduJDJ/Week3/tree/DaySix
- cd Day01
- cd mysite
- python manage.py runserver
